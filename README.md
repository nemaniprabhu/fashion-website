# 🛍️ Fashion Shopping Website

A full-stack Fashion Shopping Website built using Flask, HTML, CSS, Bootstrap, JavaScript, and SQLite.

## 🚀 Features
- User Login & Registration
- Men, Women & Kids Collections
- Add to Cart
- Wishlist
- Search Products
- Responsive Mobile Design
- Stylish Shopping UI
- Checkout System

## 🛠️ Technologies Used
- Python Flask
- HTML5
- CSS3
- Bootstrap
- JavaScript
- SQLite
